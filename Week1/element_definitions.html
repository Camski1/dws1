<p>
The p tag is used to add paragraphs/text to your web page.

This is a paragraphs.

<b>
The b tag is used to add bold text to your web page.

This is a paragraphs that end with a word using the b tag to make it bold.

<i>
The i tag is used to add italic text to your web page.

This is a paragraphs that end with a word using the i tag to make it italic.

<sup>
The sup tag is used to add information that should be raised above the standered text of your web page.
A good example of this would be coding the number, two to the power of five.

25

<sub>
The sub tag is used to add information that should be dropped below the standered text of your web page.
A good example of this would be coding H2O.

H2O

<br>
The br tag is used to add line breaks to the text on your web page.
The first line will not have a br tag, and the second will.

I would like a line break now. Did I get one?

I would like a line break now.
Did I get one?

<hr>
The hr tag is used to add a horizontal line to your web page.

Look at this cool line.

<strong>
The strong tag is used to add importance to text on your web page.
The default change in the browser will be represented by makeing the text bold.

This is a paragraphs that end with a word using the strong tag to make it strong.

<em>
The em tag is used to add emphasis to text on your web page.
When using this tag the default display will be italic.

This is a paragraphs that end with a word using the em tag to make it have emphasis.

<blockquote>
The blockquote tag is used to add a quote source to text on your web page.
When using this tag the default display will be to indent the text.

This is a paragraphs that is written as a blockquote.

<q>
The q tag is used to add quotes to text inside of your p tag.

This is a paragraphs that end with a word using the q tag to make it quotes.

<abbr>
The abbr tag is used to abbreviat text to your web page.
This can be used in the form of abbr or acronym.

A good example of this would be my initials CJK

<cite>
The cite tag is used to add a quote source to text on your web page.
When using this tag the default display will be to make the text italic.

Physics of the Future by Michio Kaku.

<dfn>
The dfn tag is used to indicate the defining instance to a new term on your web page.
The default change in the browser will be represented by makeing the text italic or none.

This is a paragraphs that end with a word using the dfn tag to make it a defining moment .

<address>
The address tag is used to add source to text on your web page.
This can be in the form of a URL, phone number, or, any other source that will work with your page.

Facebook

<ins> / <del>
The ins and del tags will add and delete text to/from your web page.
The default change in the browser will be represented by makeing ins underlined, and the del crossed out.

I don't do like to party.

<s>
The s tag is used to update information on your web page.
The default change in the browser will be represented by crossing out the text.

I am living in the state of.

Tennessee

Georgia

<ol> / <li>
The ol tag is used in conjunction with the li tag.
The ol represents an ordered list and the li represents each list item.
  1. Ham
  2. Coffee
  3. Boots
<ul> / <li>
The ul tag is used in conjunction with the li tag.
The ul represents an unordered list and the li represents each list item.
  • Ham
  • Coffee
  • Boots
<dl> / <dt> / <dd>
The dl tag is used in conjunction with the dt and the dd tags.
The dl is the container for the list,
the dt is the name of the list,
and the dd is the list items.
Shopping List
Ham
Coffee
Boots
